# Loop

```
Loop

-   Vòng lặp for thông thường có thể sử dụng ở bất kỳ đâu khi biết số lần lặp.
-  Sử dụng vòng lặp While khi không biết số lần lặp.
Cú pháp: While(đk) {
    đoạn code 
    // đk sai mới thoát vòng lặp
}
-   Vòng lặp Do While gần giống While nhưng chạy ít nhất 1 lần kể cả khi điều kiện sai.
Cú pháp: do {
    đoạn code
} while(đk)
-   for of chỉ sử dụng với Array. Dùng để lặp qua các phần tử của Array.
-   for in sử dụng để lặp qua các key của Object và các index của Array.

```
